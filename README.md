# akiBot  
基于noneBot的qq群机器人 制作中  
完成功能  
1.实时监控b站直播的功能 需要绑定直播间id 数据存储于数据库中    
2.获取每日新闻  
3.获取steam好友在线列表 需要在列表中写入steamid  
4.获取pixiv图片的功能 容易让机器人风控推荐不用  
5.获取今日天气  
功能可自取 但是请标明出处 有问题请写在issue里
